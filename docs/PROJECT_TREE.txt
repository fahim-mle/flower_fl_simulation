Key Information
===============

DEVELOPMENT LOCATIONS:
  - Source code: fl-simulation-app/fl_simulation_app/ or quickstart-docker/quickstart_docker/
  - Tests: tests/unit/, tests/integration/, tests/security/
  - Documentation: docs/ (appropriate subdirectory)
  - Configuration: configs/
  - Scripts: scripts/ (appropriate subdirectory)
  - Certificates: certificates/ (appropriate subdirectory)

GITIGNORE PATTERNS:
  - __pycache__/ - Python cache
  - *.pyc - Compiled Python
  - .fl_venv/ - Virtual environment
  - .vscode/, .idea/ - IDE files
  - .DS_Store, Thumbs.db - OS files
  - Private certificate keys - via certificates/.gitignore
  - .claude/ - AI agent configs

IMPORTANT NOTES:
  - NEVER commit private keys (*.key files)
  - Model artifacts use separate storage
  - Keep .gitignore updated for secrets
  - Update docs/README.md when adding documentation
  - Follow file placement guidelines (see FOLDER_ARCHITECTURE.md)

For full details, see: docs/FOLDER_ARCHITECTURE.md
